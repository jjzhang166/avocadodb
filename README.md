
AvocadoDB
========
v0.1

avocadodb是具有灵活的数据模型
文档、图表和k/v多结构的开源数据库。
使用Aql构建高性能应用程序
方便的sql查询语言或JavaScript扩展。


AvocadoDB的主要特点
------------------------

- **多模型**: 文档,图形，k/v  多种数据模型由你选择
- **AQL**:灵活方便的AQL查询
  **restful-api** :方便各种前后端直接调用
  **中文化的web管理界面**:不在迷茫！开箱即用！
